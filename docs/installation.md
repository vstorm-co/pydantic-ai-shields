# Installation

## Requirements

- Python 3.10 or higher
- pydantic-ai-slim >= 1.74.0

## Install with pip

```bash
pip install pydantic-ai-shields
```

## Install with uv

```bash
uv add pydantic-ai-shields
```

## Verify Installation

```python
from pydantic_ai_shields import CostTracking, PromptInjection, PiiDetector

print("Installation successful!")
```
